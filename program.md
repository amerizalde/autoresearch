# Instructions for the AI Agent

Your mission is to improve the validation loss (val_loss) of the language model in `train.py`.

## Goals
1. Minimize `val_loss`.
2. Keep the training time within the 5-minute window (do not increase `max_iters` too much).
3. Experiment with architecture (layers, heads, embedding size), hyperparameters (learning rate, dropout), and optimizer settings.

## Current Focus
Start by experimenting with the learning rate and the number of layers.
